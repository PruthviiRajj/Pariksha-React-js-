# Pariksha - Online Examination System

A React-based online examination system that allows users to take tests and submit their answers.

## Features

- User-friendly interface with a responsive design
- Multiple choice questions support
- Test timing functionality
- Real-time score calculation
- Result submission system
- Navigation between questions

## Tech Stack

- React.js 18.2.0
- React Router DOM 6.22.1
- Axios for API communication
- React Spinners for loading states
- CSS for styling

## Project Structure

```
src/
  ├── App.js          # Main application component
  ├── Home.js         # Home page component
  ├── test.js         # Test taking component
  ├── submit.js       # Submission handling component
  ├── navbar.js       # Navigation component
  ├── Card.js         # Card component
  ├── clock.js        # Timer component
  └── styles/         # CSS files for styling
```

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/PruthviiRajj/Pariksha-React-js-.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

   The application will open in your browser at [http://localhost:3000](http://localhost:3000).

## Available Scripts

- `npm start` - Runs the app in development mode
- `npm test` - Launches the test runner
- `npm run build` - Builds the app for production
- `npm run eject` - Ejects from Create React App
